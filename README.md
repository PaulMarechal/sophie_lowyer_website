# sophie_lowyer_website

Website for Sophie Maréchal Lowyer in Paris 

Website realised with React 

Forst install : 
npm i 

Test : 
npm run dev

Build :
npm run build

Calendly :
Créer un fichier `.env.local` avec :

NEXT_PUBLIC_CALENDLY_URL=https://calendly.com/votre-compte/votre-evenement
